Target product summary

AD9721

ADI marks AD9721 as obsolete and suggests AD9760 as the replacement on the current product page. Teams supporting legacy waveform-generation or communications hardware should recheck update rate, output topology, package constraints, and system timing before treating AD9760 as the validated service replacement for an AD9721-based design.
