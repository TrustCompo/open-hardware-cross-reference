Target product summary

AD1672AP

ADI lists the AD1672 family as obsolete and names AD9221 as the suggested replacement on the current product page. We published AD1672AP as the complete MPN for this obsolete 12-bit 3 MSPS monolithic ADC variant, and sustainment or redesign work should compare resolution, throughput, input architecture, supply rails, package constraints, and timing interface behavior before treating AD9221ARZ as the qualified successor.
