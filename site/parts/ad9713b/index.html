Target product summary

AD9713B

ADI currently lists AD9713B as obsolete and explicitly advises designers to consider the AD975X and AD976X product families instead. For sustainment programs, this 12-bit 80 MSPS TTL compatible DAC should be replaced only after confirming logic compatibility, output architecture, timing margins, package fit, and dynamic performance against the original AD9713B datasheet and the selected successor model.
