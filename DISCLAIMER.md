# Engineering Disclaimer

TrustCompo Open Hardware Cross Reference provides public shortlist and review
aids for lifecycle-affected electronic components.

The information in this repository is not a design approval, manufacturing
approval, legal warranty, manufacturer endorsement, or final equivalence claim.
Before using any replacement in production, engineers should validate the
candidate against the actual application, including at least:

- datasheet limits
- pinout and package
- electrical parameters
- thermal behavior
- lifecycle and availability
- compliance requirements
- qualification and reliability requirements

Use this repository to start review work faster, not to bypass engineering
validation.
